# Отчет по учебному проекту: Apple Devices Search

## 1. Цель работы
Разработать учебное веб-приложение, реализующее **поиск и сопоставление Apple-устройств** по текстовому запросу пользователя с применением алгоритмов обработки текста, эвристического ранжирования и фильтрации результатов.

---

## 2. Используемые технологии
- **Backend**: Python, FastAPI (REST API)
- **Frontend**: HTML / CSS / JavaScript
- **Среда запуска**: Docker, docker-compose

---

## 3. Структура проекта

    frontend/        — интерфейс пользователя
    backend/
      main.py        — точка входа backend-сервиса
      app.py         — создание и конфигурация FastAPI
      api/           — HTTP-роуты (эндпоинты) поиска
      matching/      — логика парсинга и сопоставления

---

## 4. Описание процесса обработки запроса

1. **Получение запроса**
   - Пользователь вводит строку поиска (например: модель, память, цена).

2. **Предобработка текста**
   - приведение к единому регистру
   - удаление/замена служебных символов
   - токенизация (разделение на слова/части)
   - удаление стоп-слов

3. **Извлечение параметров**
   - извлечение числовых значений (цена, объем памяти, год и т.п.)
   - выделение ключевых терминов (модель, линейка, модификация)

4. **Сопоставление с товарами**
   - проверка совпадений токенов запроса с полями товара:
     - `title / name / product_name`
     - характеристики
     - цена
   - учет частичных совпадений
   - учет словарей ключевых слов (синонимы/варианты написания)

5. **Ранжирование**
   - вычисление эвристического балла релевантности
   - сортировка результатов по убыванию балла

6. **Фильтрация**
   - отсев нерелевантных совпадений
   - применение ограничений по диапазонам (например, по цене)

7. **Формирование ответа**
   - возврат результата в формате JSON через REST API
   - отображение результата на фронтенде

---

## 5. Используемые алгоритмы и методы

- **Text Parsing**
  - нормализация строк
  - регулярные выражения
  - токенизация

- **Keyword Matching**
  - словари ключевых терминов
  - поиск вхождений и частичных совпадений

- **Numeric Extraction**
  - извлечение чисел из текста (цены, память)

- **Heuristic Scoring**
  - эвристическая оценка релевантности по количеству/типу совпадений

- **Stopword Filtering**
  - исключение шумовых слов из запроса

---

## 6. Ключевые модули проекта
- `parser.py` — предобработка и разбор запроса
- `matcher.py` — сопоставление запроса с товарами
- `dictionaries.py` — словари моделей/терминов/вариантов
- `filters.py` — фильтрация и стоп-слова
- `api/search` — эндпоинты поиска
